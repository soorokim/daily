2. 1을 제외한 홀수번째 대각선은 분모는 작아지고 분자는 커지는 규칙 짝수번째 대각선은 분모는 커지고 분자는 작아지는 규칙 -> 방향이 바뀐다.
3. n번째 대각선 까지 1 + 2 + 3 + 4 + n 개의 분수가 존재

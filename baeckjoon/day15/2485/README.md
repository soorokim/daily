# 문제풀기전 생각정리

1. 일단 두 점 사이의 간격을 같게 만들려면 두 점사이의 점이 2n + 1(n=0, 1,...)개 있어야 할것 같다. 그래야 중간에 가로수를 심었을때 간격이 2n이 될테니.

2. 그리고 두 점 사이의 간격중 가장 작은 값을 찾고 그값이 짝수면 그값으로 시도, 그 값이 홀수이면 2로 나눠서 몫을 갖고 시도.
